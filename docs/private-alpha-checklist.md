# ApplyPilot Private Alpha Checklist

## Before applying

- Resume selected
- Profile reviewed
- Browser opens correctly
- Application URL is valid

## During application

- Review autofilled identity information
- Review work authorization and sponsorship
- Review education and employment
- Review salary fields
- Review generated prose
- Complete CAPTCHA or login manually
- Never assume the application was submitted

## After application

- Review every page
- Submit manually
- Mark submitted in ApplyPilot
- Record corrections
- Confirm the application appears in Applications
- Add a next step when useful

## Dogfood scorecard

- Preparation time
- Corrections
- Missed required fields
- Wrong fields
- Short-answer usefulness
- Overall time saved
